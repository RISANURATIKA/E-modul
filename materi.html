1.Dasar Jaringan Komputer

Pengertian Jaringan Komputer :
Sekumpulan komputer atau perangkat IT berjumlah banyak yang terpisah-pisah akan tetapi saling berhubungan dalam melaksanakan tugasnya.

Manfaat Jaringan Komputer
Umum :

•Resource Sharing
Resource (data, program, peralatan) bisa digunakan oleh tiap orang dalam jaringan tanpa terpengaruh lokasi.
•High Reliability
Sistem dimungkinkan dapat tetap berjalan dengan memiliki sumber sumber alternative yang tersedia
•Saving Money
Computer berukuran kecil mempunyai rasio harga/kinerja yang lebih baik disbanding dengan computer yang besar
•Scalability
Kemampuan untuk meningkatkan kinerja sistem secara berangsur angsur sesuai dengan beban pekerjaan

Klasifikasi Jaringan Komputer (Jarak):

•Personal Area Network (PAN)
Interkoneksi perangkat IT dalam jarak individu (10 meter)
•Local Area Network (LAN)
Jaringan computer yang meliputi area geografis yang kecil
•Metropolitan Area Network (MAN)
Jaringan computer yang menghubungkan 2 atau lebih jaringan yang hanya terbatas pada kota
•Wide Area Network (WAN)
Mengcakup area geografis yang luas
•Internet
Interkoneksijaringan antara masyarakan umum, industry, akademis, komersial.

Klasifikasi Jaringan Komputer (fungsi):

•Client Server
Salah satu computer yang di definisikan sebagai server atau induk bagi computer lain. Server melayani computer lain yang disebut client.
•Peer to Peer
Setiap computer bisa menjadi server sekaligus client

Topologi:

•Bus
Sejumlah node (computer, peralatan jaringan) terhubung dengan jalur komunikasi bersama.
•Ring
tiap node tepat terkait dengan 2 node lainnya, membentuk jalur circular (ring). Menggunakan sebuah kabel backbone untuk transmisi data.
•Star
Tiap node dalam jaringan terhubung ke node utama/tengah.
•Tree
Gabungan topologi bus dan star. Dapat digunakan untuk menghunungkan beberapa LAN dengan LAN lain.
•Mesh
Hungungan point to point atau satu-satu ke setiap node. Jika salah satu jalur terputus pengiriman data dapat melalui rute yang lain.

2. Model Jaringan Komputer

OSI Model:

OSI Model menjelaskan bagaimana beragam perangkat jaringan dapat saling berkomunikasi satu sama lain melalui framework yang terdiri atas 7 Layer.

•Layer 7 : Applicatoin
- Interface antara jaringan dan sistem aplikasi
- Mengatur bagaimana aplikasi dapat mengakses jaringan dan kemudian membuat pesan-pesan kesalahn.
• Layer 6 : Presentation
- Mentranslasikan data yang hendak ditransmisikan oleh aplikasi ke dalam format yang dapat ditransmisikan oleh jaringan (mengatur bagaimana data dipresentasikan)
- Menangani pemrosesan seperti enkripsi, tipe data, format data, struktur data.
• Layer 5 : Session
- Mendefinisikan bagaimana koneksi dimulai, dipelihara dan diakhiri
- Singkronisasi antara pertukaran data antara komputer
• Layer 4 : Transport
- Melakukan segmentasi dan menyatukan kembali data yang tersegmentasi dari upper layer
- Flow control, memastikan host pengirim tidak mengirimkan data lebih cepat dari yang dapat diolah oleh host penerima.
- Error checking, untuk mendeteksi yang eror
- Error recovery, meminta pengiriman kembali data yang rusak/error
• Layer 3 : Network
- Menentukan alamat jaringan secara logic
- Menentukan rute yang harus diambil selama perjalanan
- Menjaga antara trafik dijaringan
- Error checking, error recovery
- Data pada bagian ini disebut paket
• Layer 2 : Data Link
- Grouping data secara logika
- Menggabungkan paket menjadi byte dan byte menjadi frame
- Menyediakan akses ke media menggunakan alamat MAC
- Melakukan pendeteksian kesalahan bukan perbaikan
• Layer 1 : Protocol
- Memindahkan bit antar devices
- Mengirim bit dan menerima bit
- Berkomunikasi langsung dengan jenis media transmisi
- Merepresentasikan bit tergantung dari media dan protocol yang digunakan
- Menentukan kebutuhan listrik, mekanis, procedural dan fungsional, mempertahankan dan menonaktifkan hunungan fisik antarsistem.

3. IP Address

Host Addressing :

• Setiap host memiliki ID
• Terdiri dari :
- Alamat Jaringan (Network Address/Network Number)
- Alamat Host (Host Address/Host Number)

IP Addressing :

• Metode pengalamatan
• Pada IPv4 memiliki 4 bagian, dengan tiap bagian terdiriatas 8 bit.
• Dalam suatu IP terdapat atas alamat jaringan dan alamat host.

Contoh Protocol/Layanan :
IP Address (dalam biner):
1010 011.1001 0000.1010 1010.0101 1010
IP Address (dalam desimal) :
163.144.170.90
Alamat IP diatas terdiri atas alamat network dan alamat host.

Subnet mask :

• Digunakan untuk mendapatkan Network Address dengan meng-AND kan dengan alamat IP suatu host.

Contoh Protocol/Layanan :
- Alamat IP = 202.46.249.33
- SubNet mask =255.255.255.0
- Network Number =202.46.249.0

Subnetting :
• Membagi suatu network menjadi sub-sub network yang lebih kecil

Broadcast Address :
• Alamat akhir dari suatu network/subnetwork
• Tidak ddapat dijadikan IP Host

Contoh Protokol/Layanan :
Network Address : 192.168.10.0
Subnet Mask : 255.255.255.224 (Biner: 11111111.11111111.11111111.11100000)
• Ada berapa subnet?
- 23 – 2 = 6 subnet
• Ada berapa host per subnet?
- 25 – 2 = 32 – 2 = 30 host per subnet
• Alamat subnet berapa saja yang valid (bisa dipakai)
- Ingat bahwa 0000 0000, dan 1110 0000 tidak bisa dipakai
